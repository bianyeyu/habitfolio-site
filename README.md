# HabitFolio Public Support Site

This folder is the source for the public HabitFolio support/privacy website.
It is intentionally isolated from the closed-source app project so it can be
published to a separate public GitHub Pages repository.

Recommended public repository:

```bash
bianyeyu/habitfolio-site
```

Recommended App Store URLs after publishing:

- Support URL: `https://bianyeyu.github.io/habitfolio-site/support/`
- Privacy Policy URL: `https://bianyeyu.github.io/habitfolio-site/privacy/`

For a custom domain later, point `habitfolio.app` to GitHub Pages and add a
`CNAME` file here before publishing.
